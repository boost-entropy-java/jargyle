Requirements

For automated testing, building, and running Jargyle under the source distribution:

  • Apache Maven 3.3.9 or higher
  • Java 9 or higher

For running Jargyle under the binary distribution:

  • Java 9 or higher

After installation of the requirements, be sure to have the environment variable JAVA_HOME set to the Java home directory.
