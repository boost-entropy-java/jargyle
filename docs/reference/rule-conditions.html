General Rule Conditions

clientAddress

Syntax:

clientAddress=ADDRESS_RANGE

Description:

Specifies the client address

Value: Address Range

socksServerAddress

Syntax:

socksServerAddress=ADDRESS_RANGE

Description:

Specifies the SOCKS server address the client connected to

Value: Address Range
